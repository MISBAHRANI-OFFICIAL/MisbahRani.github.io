Electric Air Purification System for Particulate Matter

A smart solution for real-time air quality monitoring and filtration

Contact the Team

Overview

The Electric Air Purification System for Particulate Matter is a smart air purifier designed to monitor and filter harmful particles like PM₂.₅ and PM₁₀ in real time. Developed by a team of electrical engineering students, the system integrates a microcontroller, high-precision sensors, and an adaptive filtration mechanism to provide an efficient and affordable solution for indoor air quality management.

Figure 1: System overview diagram showing main components

Key Components and Assembly

1. Microcontroller (Central Processing Unit)

  • Options: Tiva TM4CH6PM or ESP32 microcontroller
  • Role: Acts as the brain of the system, processing sensor data and controlling the fan and filtration system
  • Communication: Uses I²C protocol for efficient data transfer between components

Figure 2: Tiva and ESP32 microcontrollers used in the project

2. Sensor Module

  • Sensor: SN-GCJA5 laser-based particulate matter sensor
  • Function: Continuously monitors air quality by detecting PM₂.₅ and PM₁₀ concentrations
  • Data Handling: Sends real-time data to the microcontroller for analysis and response

Figure 3: SN-GCJA5 laser PM sensor in operation

3. Filtration System

  • Filter Material: Activated charcoal-coated car AC filters
  • Mechanism: The adsorbent layer captures PM₂.₅ and PM₁₀ particles as air passes through
  • Testing: Initial trials used car AC filters to evaluate adsorption efficiency before finalizing the design

Figure 4: Activated charcoal-coated filter assembly

4. Fan Control

  • Adjustable Fan: Dynamically changes speed based on air quality readings
    • High Pollution: Increases fan speed for intensive filtration
    • Low Pollution: Reduces speed to save energy
  • Integration: Controlled by the microcontroller using real-time sensor feedback

Figure 5: Variable speed fan mechanism

5. User Interface

  • Platform: Web-based interface accessible via Wi-Fi or Bluetooth
  • Features:
    • Displays live air quality data (PM₂.₅, PM₁₀ levels)
    • Allows remote control of purifier settings (fan speed, on/off)

Figure 6: Web-based user interface for monitoring and control

System Workflow

  1. Detection: The SN-GCJA5 sensor detects particulate matter levels in the air
  2. Data Processing: The microcontroller analyzes the sensor data and determines the required fan speed
  3. Filtration: Polluted air is drawn through the charcoal-coated filter, where particles are adsorbed
  4. User Feedback: Real-time air quality metrics are displayed on the web interface, with options for manual override

Figure 7: Complete system workflow diagram

Testing and Evaluation

  • Performance: Tested in varied environments to assess filtration efficiency and power consumption
  • Benchmarks: Compared against industry standards to ensure compliance
  • Outcome: The system demonstrated responsive adaptation to pollution levels while maintaining energy efficiency

Figure 8: System under testing in controlled environment

Conclusion

This project delivers a cost-effective, smart air purifier that addresses indoor air pollution through real-time monitoring and automated filtration. By leveraging affordable materials (e.g., repurposed car filters) and IoT-enabled controls, the team aims to make air quality management accessible in regions like Pakistan, where pollution poses significant health risks.
